# AGENTS.MD — Карта агентов и ролей

Статус: активный  
Последнее изменение: [06 апреля 2026, 20:00]

## Агенты проекта

| Агент | Роль | Зона ответственности |
|-------|------|---------------------|
| OpenCode | Исполнитель | Код, файлы, тесты |
| ZeroClaw | Оркестратор | Планирование, маршрутизация |
| Claude Code | Архитектор | Архитектурные решения |
| Telegram Bot | Интерфейс | Удалённое управление |

## Правила взаимодействия

1. Все агенты читают AI.MD перед началом работы
2. Состояние синхронизируется через `.state/`
3. Память общая через Supermemory MCP
4. Handoff при завершении сессии
