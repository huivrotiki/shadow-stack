# сделай  графический план диаграму

Вот графический план Shadow Stack v3.2 в виде диаграмм (формат Mermaid, их можно вставить в Obsidian/Notion/GitHub). Схема основана на твоём гайде по Shadow Stack на Mac M1.[^1]

***

## 1. Слои архитектуры

```mermaid
flowchart TB
    subgraph ControlPlane["Control Plane"]
        TG[Telegram / owpenbot]
        ZC[ZeroClaw Gateway]
    end

    subgraph IntelligencePlane["Intelligence Plane"]
        AR[Shadow Auto-Router]
        OL[Ollama (Qwen / Phi)]
        OR[OpenRouter Free Tiers]
        BR[Browser Automation\n(Playwright MCP / n8n)]
        CLP[Claude Paid (по запросу)]
    end

    subgraph ExecutionPlane["Execution Plane"]
        VC[Vercel CLI / Deploy]
        OC[OpenCode IDE / Claude Code]
        AG[Antigravity IDE (опция)]
    end

    subgraph MacOSPlane["macOS Automation"]
        AS[AppleScript]
        AM[Automator Workflows]
        SC[Shortcuts / Shadow Start]
    end

    TG --> ZC
    ZC --> AR

    AR --> OL
    AR --> OR
    AR --> BR
    AR --> VC
    AR --> CLP

    ZC --> OC
    ZC --> AG

    VC --> AM
    AM --> AS
    SC --> AS
```


***

## 2. Поток запроса из Telegram

```mermaid
sequenceDiagram
    participant User as Пользователь в Telegram
    participant TG as Telegram Bot (owpenbot)
    participant ZC as ZeroClaw
    participant AR as Auto-Router
    participant OL as Ollama
    participant BR as Браузер (Playwright/n8n)
    participant VC as Vercel CLI

    User->>TG: /ask "Сделай лендинг"
    TG->>ZC: Webhook /update
    ZC->>AR: Текст сообщения

    AR->>AR: Классификация (Qwen3B)\nlocal / smart / code / browser / deploy / premium

    alt Простой / короткий запрос
        AR->>OL: prompt
        OL-->>AR: ответ
    else Сложный / длинный запрос
        AR->>BR: HTTP → /webhook/llm
        BR-->>AR: текст из Gemini/Claude
    else /deploy
        AR->>VC: vercel deploy --prod
        VC-->>AR: URL деплоя
    end

    AR-->>ZC: итоговый ответ
    ZC-->>TG: текст
    TG-->>User: ответ в чат
```

## 3. Этапы (Phase 1→5) как дорожная карта

```mermaid
gantt
    dateFormat  YYYY-MM-DD
    axisFormat  %d.%m

    section Phase 1: Router + Telegram
    Auto-router.ts                :a1, 2026-03-22, 1d
    SKILL.md routing rules        :a2, after a1, 0.5d
    Telegram webhook /api/...     :a3, after a2, 0.5d

    section Phase 2: CI/CD + macOS
    GitHub Actions deploy.yml     :b1, after a3, 0.5d
    shadow-start.sh               :b2, after b1, 0.5d
    Automator shadow-deploy       :b3, after b2, 0.5d
    Shortcut "Shadow Start"       :b4, after b3, 0.2d

    section Phase 3: OpenCode
    oh-my-opencode + плагины      :c1, after b4, 1d

    section Phase 4: Observability
    SSE /api/logs/stream          :d1, after c1, 0.5d

    section Phase 5: Docs
    RUNBOOK.md / AGENTS.md        :e1, after d2, 0.5d

```
